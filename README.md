#  SiteBuilder – AI Powered Website Generator (PERN Stack)

SiteBuilder is a full-stack SaaS web application that allows users to generate complete, production-ready websites using AI.  
Users can create projects, preview versions, manage revisions, publish websites, and purchase credits via Stripe.

---

 

---

##  Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS
- Axios
- Better Auth

### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL (Neon DB)

### Payments
- Stripe Checkout
- Stripe Webhooks

### AI Integration
- OpenAI / KwaiPilot Model API

---

##  Features

-  Authentication & Session Management
-  AI Website Generation
-  Project Version Control
-  Live Preview Rendering
-  Publish & Community Sharing
-  Stripe Credit Purchase System
-  Credit-based AI generation system

---

##  How It Works

1. User logs in
2. User enters website prompt
3. AI enhances prompt
4. AI generates full Tailwind-based HTML website
5. Project versions are saved
6. User can preview, edit, publish, or rollback
7. Credits are deducted per generation
